## Installation

How to run Codeigniter

```bash
    Windows(Using Xampp)
    put file to .htdocs
    start Apache & mysql
    to access you can using localhost
    example:
    (Postman)
    http://localhost/toko_api/public/registrasi
    (PHPMyAdmin)
    https://localhost/phpmyadmin
```

How to Run Flutter
```bash
    flutter run
```
